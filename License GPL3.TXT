Preâmbulo
A Licença Pública Geral GNU (GPL) é uma licença de software livre que garante aos usuários a liberdade de usar, modificar e distribuir software. Ela também garante que o software permaneça livre para sempre, impedindo que qualquer pessoa o torne proprietário.
Esta licença é baseada na filosofia do software livre. Acreditamos que os usuários de software devem ter a liberdade de executar, copiar, distribuir, modificar e estudar o software que usam. Também acreditamos que o software deve ser livre para ser redistribuído, com ou sem modificações, para que todos possam se beneficiar dele.

"Código-fonte" significa a forma preferencial do Software para modificá-lo. Para a maioria dos softwares, isso corresponde ao código-fonte em linguagem de programação.
